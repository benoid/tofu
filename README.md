tofu
====

`raco pkg install https://github.com/benoid/tofu.git`

An implementation of a bf-like language in Racket.
Source code must be neatly cubed.

```
read byte:          ,
write byte:         .
increment byte:     ^
decrement byte:     _
move pointer left:  |
move pointer right: I
comment begin/end:  `
```



Hello, World in tofu
